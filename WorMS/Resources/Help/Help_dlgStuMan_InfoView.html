View/Edit Student Information

This dialog is used to enter information to add a new student or to view or change the information of an existing student. This information includes their name, level, whether they have paid, whether they have had a receipt printed and what resources they have used.

  1. Name text box - used to add or view a student's name; once a student's name is saved, it is permanent and cannot be changed unless a new student is created to replace the old student record
  2. Level selector - used to select the level the student is in their A-level studies
  3. Cost owed text box - used to enter the cost a student owes; this value can be 0 if the student owes no money
  4. Paid radio buttons - used to select whether the student has paid or not
  5. Receipt printed radio buttons - used to select whether the student has had their receipt printed or not; this value is handled automatically but can be changed manually if needed
  6. Resources list box - used to list the resources a student has used, based from entry through the Resource Manager
  7. Clear button - used to deselect the current resource selection on the list
  8. Remove button - used to remove the resource selected on the list
  9. OK button - used to enter the input into the program and perform the action selected earlier
  10. Cancel button - used to cancel any input entered; the action selected earlier will not be taken


Usage

When a student is to be added, the parts on the top of the dialog can be used to add the student's name, level, cost owed, whether they have paid and whether they have had their receipt printed. Resources can only be added to the student's record through the Resource Manager, once the student has been added.

When a student is to be changed, any of the information can be changed except for the student's name. Resources in the resource list can be selected and removed if needed, by clicking the Remove button below the list.

Upon clicking View Info for a selected student in the Student Manager, the student's information can only be viewed through the dialog.

Clicking Cancel at any time will return the user to the Student Manager and no actions will be applied concerning information input. Clicking OK will mean the information input will apply as selected.
