Home Screen

This form is used as a link to all functionalities throughout the program. Different tasks that can be completed with the program are split into separate forms, which can be opened by simply clicking one of these buttons.

  1. Manage Resources - Opens the Resource Manager so resource stock can be viewed and manipulated
  2. New Requisition Form - Opens the Requisition Form Editor so a new requisition form can be produced
  3. View Saved Requisition Forms - Opens the Requisition Form Manager so existing requisition forms can be viewed and manipulated
  4. Manage Suppliers - Opens the Supplier Manager so suppliers can be viewed and manipulated
  5. Manage Students - Opens the Student Manager so students, the resources they have used and the cost they owe can be viewed and manipulated
  6. Manage Machine Test Records - Opens the Machine Test Records Manager so records concerning testing of machines in the workshop can be viewed and manipulated
  7. Change Data Directory - Opens the Change File Directory dialog so the location for storing data used by the program can be viewed and changed
  8. Quit - Closes the program
