View Low Stock

This dialog is used to view resource stock low in quantity, displaying the amount each resource is under their minimum stock. The list of resources low in stock may be printed and resources in the list may be selected to add to a new requisition form.

  1. Low stock resources list - used to display and give the ability to select resources low in stock quantity
  2. Add Selected to New Form button - used to take the resources selected in the list and add them to a new requisition form
  3. Print button - used to generate a print preview of all resources in low stock
  4. Close button - closes the dialog and returns the user to the Resource Manager


Usage

When the dialog is opened, the resource list will automatically populate with resources low in stock in the order of category first and then alphabetically within each category. A resource is considered low in stock when its stock amount reaches or falls below its minimum stock range. The list displays each resource as an item, with its current amount, minimum amount and the amount its current stock is below the minimum.

Clicking the Print button will generate a print preview of all resources low in stock, which can then be printed for reference or archiving purposes.

Multiple low stock resources can be selected by clicking each of them individually, and deselected by clicking on them once again.

Doing so will enable the Add Selected To New Form button, where clicking it will create a new requisition form with the resources you have selected and display it on screen in the Requisition Form Editor.
