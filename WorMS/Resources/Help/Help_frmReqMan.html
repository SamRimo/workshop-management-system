Requisition Form Manager

This form is used to manage saved requisition forms. The type of form can either be a draft, unprinted final or printed final form, all of which have their own unique differences.

  1. Select type radio buttons (printed, unprinted, drafts) - used to select a certain type of form and display it in the requisition form list
  2. Requisition form list - used to display the requisition forms saved under the category selected above; forms are always sorted in chronological order
  3. Reuse/Change button - used to open the currently selected requisition form for editing through the Requisition Form Editor
  4. Export to Excel button - used to copy the contents of the currently selected requisition form to an Excel spreadsheet formatted for printing as an actual requisition form; this will only work with unprinted final or printed final forms
  5. Remove button - used to delete the currently selected requisition form


Usage

Upon clicking Drafts as the requisition form type, a list of all the saved draft forms will generate.

Clicking a form in the list will enable the Change and Remove buttons below, which let you make changes to the selected form or remove it completely. Draft requisition forms cannot be exported to Excel and must be saved again as final to do so.

NOTE: When a form is saved again after opening it in the Requisition Form Editor, the original form will be kept alongside the new one created. Items in the list will always be sorted in chronological order so the old form can be easily found in the list, selected and deleted manually.

Upon clicking Unprinted as the requisition form type, a list of all the saved unprinted final forms will generate.

Clicking a form in the list in this mode will enable all the buttons below, allowing you to not only reuse or remove the selected form but also export it as an Excel document, formatted as a requisition form for actual use. Just like with draft forms, after selecting Reuse and saving the form in the Requisition Form Editor again, it will create a copy of that form alongside it.

Clicking Export to Excel will trigger Microsoft Excel to generate a formatted requisition form, which will be saved to the user's documents folder. If the form was exported successfully, a message box will open which also states the location where it was saved.

NOTE: Your computer must have Microsoft Excel installed to do this. If you do not have it installed, the program will likely crash.

The exported form can then be viewed by opening the file through Microsoft Excel. For parts that may not fit properly, such as the item name in this case, the font size can be changed manually to accompany this.
