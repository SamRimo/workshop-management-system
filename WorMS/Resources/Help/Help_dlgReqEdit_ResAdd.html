Add/Edit Requested Resource

This form is used to add resources to a requisition form through the Requisition Form Editor.

  1. Catalogue number text box - used to enter the catalogue number for a new resource or otherwise view or edit an existing one
  2. Item name text box - used to enter the item name for a new resource or otherwise view or edit an existing one
  3. Cost per piece text box - used to enter the cost for each piece of a new resource or otherwise view or edit an existing one
  4. Quantity spin box - used to enter the quantity wanted of a new resource or otherwise view or edit an existing one
  5. Discount text box - used to enter the discount given to the cost of each piece of a new resource or otherwise view or edit an existing one
  6. VAT text box - used to enter the VAT given to the cost of each piece of a new resource or otherwise view or edit an existing one
  7. OK button - used to add a new resource to the requisition form or change the information for an existing one
  8. Cancel button - used to close the dialog and clear any input entered; no resources will be added or changed


Usage

Simply enter the information required for a resource into each of the appropriate input boxes and click OK when finished.

NOTE: None of the input boxes should be empty when clicking OK, otherwise it is likely to be invalid. To input an empty discount and/or VAT amount, it must be given the value of 0 for the resource entry to pass as valid.

If a resource is being changed, its existing data will be imported and can be changed as appropriate.

Clicking OK when changing a resource will modify its information as given.
