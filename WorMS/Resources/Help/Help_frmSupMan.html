Supplier Manager

This form is used to manage suppliers for usage in requisition forms and track their usage.

  1. Name text box - used to view or enter a name of a supplier
  2. Times used spin box - used to view or enter the number of times a supplier has been used
  3. Address text box - used to view or enter a supplier's address
  4. Add/Change button - used to add a new supplier with the information entered or to change the information of one selected in the list
  5. Clear button - used to clear the supplier list selection and any data entry
  6. Remove button - used to remove the currently selected supplier
  7. Supplier list box - used to select a supplier already on file to view or change its contents or delete it altogether; suppliers are always in decreasing order of times used


Usage

On opening the Supplier Manager for the first time or after changing the folder location, you are likely to be presented with an empty window:

The information in the list box to the right is to indicate that the program has found no records on the suppliers file.

NOTE: If the suppliers file was to be deleted or tampered with, some or all records on the actual file could be ignored. Please see Troubleshooting for more details if suppliers you have previously saved cannot be found.

To add your first supplier to list, simply input the supplier name and address in the boxes on the left and click Add when finished. The times used can be changed if you wish but this optional as it will be handled by the program automatically.

Selecting a record in the list will make its contents appear in the boxes on the left, where it can be viewed and changed by making changes and clicking the Change button. The supplier may also be removed by clicking the Remove button.

Any time an unprinted final requisition form is exported to Excel through the Requisition Form Manager to become a printed final form, the times used of the supplier associated with it will increment and the suppliers will sort as appropriate to represent the supplier 'hierarchy'. If a form has already been printed, the times used for that supplier will not update further.
